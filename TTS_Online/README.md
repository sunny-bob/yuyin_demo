# TTS_Online

#### 介绍
讯飞语音SDK例子简单集成,包含语音听写、语法识别、语义理解、语音合成、语音评测、语音唤醒、声纹密码、人脸识别等功能

#### 使用说明

1. 先去官网注册账号，创建自己的应用获取App_id
2. 下载相应的sdk
3. 解压到本地运行，注意libmsc.so的路径，App_Id一定要和官网注册的一致，否则会报10407错误
